FAQ
Frequently Asked
Questions.
Which UT standards does Scan-Master support?

AMS-STD-2154E (all classes AAA through C), ASTM A388, ASTM E2375, BS EN 10228-3/4, AMS 2631, and several Pratt & Whitney NDIP specifications. The system auto-fills acceptance criteria, FBH sizes, and equipment requirements for each standard.

How does automatic calibration block selection work?

When you enter part geometry and dimensions in the Setup tab, the system maps your part type to a geometry group, checks wall thickness, diameter ratios, and material properties, then recommends the optimal calibration block type with a detailed explanation.

Can I use Scan-Master offline?

Yes. The Electron desktop app runs fully offline with a local PostgreSQL database. Updates can be delivered via USB for air-gapped factory environments. No internet connection required for any core functionality.

What export formats are supported?

PDF (technique sheets and inspection reports), DOCX (Word documents), DXF (AutoCAD-compatible drawings), SVG and PNG (technical drawings), and STEP-compatible 3D geometry output.

How many part geometries are supported?

28+ part types including plates, blocks, cylinders, tubes, rings, disks, forgings, impellers, blisks, HPT disks, flanges, elbows, structural profiles (I/L/U/T/H), and custom geometries with uploaded drawings.

What scan directions does the system generate?

Up to 12 scan directions (A through L) based on the part geometry. These include axial, radial, and circumferential directions, plus angle beam paths calculated using Snell's Law for the specific material velocity.

Which platforms does Scan-Master run on?

Desktop: Windows, macOS, and Linux via Electron. Server: Docker, AWS (Elastic Beanstalk, ECS, Lambda), and GCP (App Engine). The system is built as a web application so it can also run in any modern browser.

Does the 3D viewer work with all part types?

Yes. Every supported geometry renders in real-time 3D using Three.js with PBR materials and studio lighting. You can rotate, zoom, and inspect parts interactively. Scan directions are visualized as overlay arrows on the 3D model.
