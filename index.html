Auto Calibration Block

AI engine maps part geometry → dimensions → material to the optimal calibration block. Returns confidence score and fallback alternatives.

// Auto-recommendation result
block: "cylinder_notched"
confidence: 95%
reason: "Thin wall <25mm, t/D <0.1"
fallback: "cylinder_fbh → flat_fbh"

Coverage Calculator

Physics-based beam modeling: near-field distance, beam divergence, sensitivity profiles. Optimizes scan index for 100% volumetric coverage.

// N = D² × f / (4 × v)
nearField: 34.2 mm
beamDiv: 4.7°
coverage: 100% "✓ Full"
deadZone: 0.5 mm (immersion)

DAC & TCG Curves

Distance-Amplitude Correction with material attenuation physics. Frequency-scaled coefficients. Time-Corrected Gain auto-generation.

// DAC = Ref - DistLoss - αΔd
material: "steel" 0.008 dB/mm
freq_scale: ×1.0 @ 5MHz
transfer_corr: ±6 dB max

Angle Beam Physics

Full Snell's Law implementation for wedge calculations. Critical angle detection, skip distance, and sound path for 45°/60°/70° beams.

OEM Rule Engine

Vendor-specific rules auto-applied: Pratt & Whitney NDIP specs enforce 5 MHz frequency, approved transducers, DAC curves, and ±2 dB transfer correction.
